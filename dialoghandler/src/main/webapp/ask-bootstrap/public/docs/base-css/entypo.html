How to use

All icons require an <span> tag with class name 'entypo' and HTML value of the character in the span tag. To use, place the following code just about anywhere:

<span class="entypo">🔔</span>

Icon examples

Use them in buttons, button groups for a toolbar, navigation, or prepended form inputs.

Buttons

Button group in a button toolbar
✉ 🔿 ✎ 
<div class="btn-toolbar">
  <div class="btn-group">

    <a class="btn" href="#"><span class="entypo">✉</span></a>
    <a class="btn" href="#"><span class="entypo">🔿</span></a>
    <a class="btn" href="#"><span class="entypo">✎</span></a>
  </div>
</div>
